# 💬 AI Text Generator App

A web application built using **Streamlit** and Hugging Face's **Transformers** library to generate natural language text using the `distilgpt2` model.

## 🚀 Features
* Clean interactive web UI via Streamlit.
* Automated text continuation utilizing deep learning transformers.
* Predefined prompt selection dropdown for quick deployment testing.

## 🛠️ Installation & Running
Make sure you have your dependencies ready, then execute:
```bash
pip install streamlit transformers torch
streamlit run project1.py
```
